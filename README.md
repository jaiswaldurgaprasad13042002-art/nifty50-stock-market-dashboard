# NIFTY 50 — Stock Market Performance Dashboard

## Project Overview
Interactive financial dashboard analyzing 10 NIFTY 50 stocks 
over 21 years (January 2000 to April 2021) using 44,219 rows 
of historical data.

**Tools Used:** Microsoft Excel / Google Sheets + ChatGPT

---

## Key Findings
- **Best Performer:** HINDUNILVR — 11.90x return over 21 years
- **Worst Performer:** WIPRO — lost 82% of investment value
- **4 out of 10 stocks** destroyed investor wealth long term
- **Safest stock:** HINDUNILVR — lowest volatility at 24.3%
- **Most volatile:** TATAMOTORS — 42% average volatility
- **Best diversification pair:** INFY + ITC (correlation = 0.08)
- **COVID crash:** HINDUNILVR was the only stock that went UP 
  during March 2020 (+10.78%)
- **2008 GFC:** TATAMOTORS lost -79% in 2008 then gained 
  +364% in 2009 recovery

---

## Dashboard Features
- 6 KPI Scorecards
- Interactive Symbol and Year filters
- Line chart — Monthly price movement (2000–2021)
- Moving average chart — Close vs MA10 / MA20 / MA50
- Bar chart — Cumulative return comparison all 10 stocks
- Correlation matrix — 10×10 heatmap with color scale
- Performance leaderboard with Sharpe ratio and verdict
- Risk vs Return scatter chart

---

## Financial Metrics Calculated
- Daily Return %
- Cumulative Return
- Moving Averages (10, 20, 50 day)
- Volatility (20 day rolling standard deviation)
- Correlation matrix
- Sharpe Ratio

---

## Dataset
- **Source:** Kaggle — NIFTY 50 Historical Stock Data
- **Stocks:** AXISBANK, HDFCBANK, HINDUNILVR, INFY, ITC, 
  RELIANCE, SBIN, TATAMOTORS, TCS, WIPRO
- **Period:** January 2000 — April 2021
- **Rows:** 44,219
